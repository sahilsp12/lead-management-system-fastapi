# AI Usage Disclosure

This document discloses the utilization of AI tools in the development of the Mini Lead Management System.

### Scope of AI Assistance
AI was used to:
1. **Structure Layouts**: Scaffold and format the standard FastAPI router directory structure and create the React pages boilerplate.
2. **Boilerplate & Routing**: Generate standard React Route switch paths and Axios request/response interceptor setups.
3. **Database Schema Design**: Recommend relational mappings for User, Lead, and ActivityLog tables.
4. **Documentation**: Help draft the Markdown ER diagrams and code architecture instructions.

### Human Developer Review & Customization
All components, logic (such as the Round Robin assignment algorithm), database queries, and styling files were reviewed and adjusted by the developer to ensure simplicity, correctness, and alignment with the assignment goals.
